# MLCodes
Set of few basic ML algorithms implemented and tested in python as a beginner practice.

# Algorithms used:
1. [KNN algorithm](https://en.wikipedia.org/wiki/K-nearest_neighbors_algorithm)
   
2. [K-Mean algorithm](https://en.wikipedia.org/wiki/K-means_clustering)
   
3. [Decission Tree algorithm](https://en.wikipedia.org/wiki/Decision_tree)

4. [Navie Bayes algorithm](https://en.wikipedia.org/wiki/Naive_Bayes_classifier)

5. [Linear Regression](https://en.wikipedia.org/wiki/Linear_regression)

6. [Suppot Vector Machine](https://en.wikipedia.org/wiki/Support_vector_machine)
